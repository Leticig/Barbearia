## 🎨 Dicas para Imagens

1. **Qualidade:** Use imagens de boa qualidade (mínimo 72 DPI)
2. **Formato:** Prefira JPG para fotos e PNG para imagens com transparência
3. **Otimização:** Comprima as imagens para carregamento rápido
4. **Direitos:** Use apenas imagens que você tem direito de usar

---

## 🔄 Como Trocar as Imagens

1. **Renomeie** suas imagens conforme a lista acima
2. **Copie** as imagens para esta pasta (`/images/`)
3. **Substitua** as imagens antigas pelas novas
4. **Atualize** a página do navegador (Ctrl+F5)

## ❓ Problemas?

Se a imagem não aparecer:
1. Verifique se o nome do arquivo está correto
2. Verifique a extensão (.jpg ou .png)
3. Limpe o cache do navegador (Ctrl+F5)
4. Verifique se a imagem está na pasta correta
